## 关键约束

1. **lark-oapi 1.5.5** 官方 SDK，使用 WebSocket 长连接模式（无需公网 IP）。
2. **自动重连**: 断线后指数退避重连（1s -> 2s -> 4s -> ... -> 60s 封顶）。
3. **卡片大小预检**: 发送前检查 `len(json.dumps(card).encode("utf-8")) / 1024`，超过 25KB 记录 warning 日志。
4. **股票代码识别规则**:
   - A股: 6 位纯数字（000001、600519、300750）
   - 港股: 5 位数字 + ".HK"（00700.HK、09988.HK）
   - 中文简称: 2-6 个汉字（如"茅台"、"平安银行"）—— 本期仅做提取，不实现模糊匹配（FEISHU-06 为 v2）
   - 无效代码: 不匹配上述任何规则
5. **免责声明固定文本**: "AI生成内容仅供参考，不构成投资建议"（COMP-01 要求）。
6. **数据截止时间格式**: "数据截止 HH:MM"（DELIVERY-04 要求）。

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Spoofing | gateway.py WebSocket 连接 | mitigate | 使用 lark-oapi 官方 SDK，SDK 内部已做签名验证 |
| T-02-02 | Tampering | message_parser.py 用户输入 | mitigate | 正则提取股票代码，拒绝非预期格式；不执行用户输入 |
| T-02-03 | Repudiation | card_sender.py 消息发送 | accept | 个人 demo 场景，飞书服务器记录发送日志即可 |

| T-02-06 | Elevation of Privilege | gateway.py 事件处理 | accept | 单用户 demo，无多租户权限隔离需求 |
