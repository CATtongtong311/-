### 3.1 沪深股市

- **接口地址**：`https://web.juhe.cn/finance/stock/hs`
- **请求方式**：`GET`
- **请求示例**：`https://web.juhe.cn/finance/stock/hs?gid=sh601009&key=您的APPKEY`

## 四、错误码说明

### 4.1 服务级错误码（`error_code`）

| 错误码 | 说明 |
|--------|------|
| `202101` | 参数错误 |
| `202102` | 查询不到结果 |
| `202103` | 网络异常 |

### 4.2 系统级错误码（`error_code`）

| 错误码 | 说明 | 旧版本对应 `resultcode` |
|--------|------|------------------------|
| `10001` | 错误的请求KEY | `101` |
| `10002` | 该KEY无请求权限 | `102` |
| `10003` | KEY过期 | `103` |
| `10004` | 错误的OPENID | `104` |
| `10005` | 应用未审核超时，请提交认证 | `105` |
| `10007` | 未知的请求源 | `107` |
| `10008` | 被禁止的IP | `108` |
| `10009` | 被禁止的KEY | `109` |
| `10011` | 当前IP请求超过限制 | `111` |
| `10012` | 请求超过次数限制 | `112` |
| `10013` | 测试KEY超过请求限制 | `113` |
| `10014` | 系统内部异常 | `114` |
| `10020` | 接口维护 | `120` |
| `10021` | 接口停用 | `121` |

> **错误码格式说明**（示例：`200201`）：
> - 第1位：`2` 服务级错误，`1` 系统级错误
> - 第2-4位：服务模块代码（即数据ID）
> - 第5-6位：具体错误代码

## 六、快速参考代码示例

### Python 请求示例

```python
import requests

APP_KEY = "0caf4e62da0681823c4fd9453de72589"
BASE_URL = "https://web.juhe.cn/finance/stock"

# 1. 查询沪深单只股票
def get_hs_stock(gid):
    url = f"{BASE_URL}/hs"
    params = {"gid": gid, "key": APP_KEY}
    resp = requests.get(url, params=params)
    return resp.json()

# 2. 查询港股
def get_hk_stock(num):
    url = f"{BASE_URL}/hk"
    params = {"num": num, "key": APP_KEY}
    resp = requests.get(url, params=params)
    return resp.json()

# 3. 查询美股
def get_usa_stock(gid):
    url = f"{BASE_URL}/usa"
    params = {"gid": gid, "key": APP_KEY}
    resp = requests.get(url, params=params)
    return resp.json()

# 4. 获取股票列表（沪股/深股/港股/美股）
def get_stock_list(endpoint, page=1, stock=None):
    """
    endpoint: shall(沪股), szall(深股), hkall(港股), usaall(美股)
    """
    url = f"{BASE_URL}/{endpoint}"
    params = {"key": APP_KEY, "page": page}
    if stock:
        params["stock"] = stock  # a=A股, b=B股
    resp = requests.get(url, params=params)
    return resp.json()

# 示例调用
if __name__ == "__main__":
    # 查询南京银行
    result = get_hs_stock("sh601009")
    print(result)
```
